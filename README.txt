然后有一个非常恶心的内容，就是那个component文件夹，里面全是组件。组件就是比如说这里面有一个btn下面的IConBtn.vue，这玩意就定义了好像是一个条形的东西。这玩意就很恶心。因为这些组件是唯一的。你要想该动这个框里面的内容就必须要动这些组件。但是组件这玩意就和函数差不多，他不只是这一个页面要用能理解吗。组件的更改一定要慎重。
然后下面的views目录下面的.vue应该是页面布局，这玩意是唯一的。应该不会影响到别人。但是不改组件的话，就比如说现在有一张白纸。这个白纸的左上四分之一位置有一幅画。这个画就是组件。不改组件意味着你可以在这张纸的空白位置加其他东西。也可以整体挪动这幅画的地点，比如放到左下四分之一。但是画的内容你永远改不了。这玩意……总之慎用。因为组件可以被多次调用。
判断调用就比如说还是这个IconBtn吧。这玩意在views 的general下面的这个JudegSever，这应该是个页面。这个里面的69行有个icon-btn标签（也许是标签？）这玩意就是在调用上面那个Iconbtn文件（或许是一部分？）总之更改慎重。不过这种组件个人感觉不用更改或许？这里面有没有位置参数我不确定。如果有的话那最好改。因为这玩意优先级挺高的。就是你页面文件里面的东西可能挤不动他，就是这个意思。总之看情况吧。慎重哦亲。
